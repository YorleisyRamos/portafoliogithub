Hola, mi nombre es Yorleisy Ramos Mosquera

Frontend Developer

Mi enfoque en el desarrollo de software, especialmente en la parte del frontend, se basa en crear la lógica y las funcionalidades que sostienen una aplicación, en lugar de concentrarme en el diseño o la interacción directa con el usuario. Esto implica trabajar en la estructura interna del software, gestionar datos, garantizar la seguridad y optimizar el código. Los lenguajes que utilizamos para dar forma y estilo al código, así como para facilitar la interacción con los usuarios, son principalmente HTML, CSS y JavaScrip - HTML: Es el lenguaje estándar para construir páginas web. Se usa para definir la estructura y el contenido de una página mediante etiquetas que describen cómo deben mostrarse los elementos. - CSS: Se emplea para definir el estilo y la apariencia de una página web escrita en HTML. Permite controlar aspectos como colores, fuentes, diseño y espaciado de los elementos HTML. - JavaScript: Es un lenguaje de programación que añade interactividad y dinamismo a las páginas web. Facilita la validación de formularios, la creación de animaciones, la manipulación del DOM y la comunicación con servidores..
